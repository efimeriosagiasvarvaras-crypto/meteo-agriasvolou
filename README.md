# Meteo Αγριάς

## Εκκίνηση
Από το PowerShell ή Terminal μέσα στον φάκελο του project:

```bash
npm install
npm run dev
```

Μετά άνοιξε στον browser:

```text
http://localhost:3000
```

Από κινητό στο ίδιο Wi-Fi:
1. Βρες το IPv4 του υπολογιστή με `ipconfig`
2. Άνοιξε:
   `http://TO-IP-SOU:3000`

## Σημειώσεις
- Η εφαρμογή χρησιμοποιεί προσωρινά το δημόσιο dashboard του σταθμού.
- Μόλις δοθεί token, μπορεί να μετατραπεί σε πλήρως αυτόνομη εφαρμογή.
